# 🦷 Eva Dent – Dental Clinic Branding Website

## 🌟 Overview

**Eva Dent** is a modern dental clinic branding and web platform designed to present professional dental services in a clean, trustworthy, and user-friendly way.

The project focuses on:

* Strong medical branding
* Modern UI/UX
* Clear service presentation
* Patient trust & credibility
* Responsive design for all devices

This project can be used as:

* A clinic presentation website
* A landing page for marketing campaigns
* A base template for dental SaaS systems

---

## 🎯 Project Goals

* Build a premium dental clinic brand identity
* Present services clearly and professionally
* Increase patient conversion rate
* Ensure mobile-first responsiveness
* Maintain clean and scalable architecture

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/kauroah/dentistry_eve.git
cd dentistry_eve
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```

Then open:

```
http://localhost:3000
```

---

## 📦 Build for Production

```bash
npm run build
npm start
```
